  \subsection{GFT: Google Fusion Tables}

  \colorbox{xxlightblue}{\vbox{%%  
  GFT is a service that [in theory] allows you to impress f\&f without
  the need to spend too much time to learn the principles of spatial
  data visualization.}}

  Limits: 1 GB / user; 350,000 map features, maybe some other...
  
  The data is stored in tables (as in a spreadsheet). The table may
  contain 4 types of data: NUMBER, STRING, LOCATION, DATETIME.

  Values in the LOCATION column can be strings with an address or pair
  latitude/longitude or KML document fragment specifying a point,
  line, or polygon.

  LOCATION columns allows data presentation on Google map.
    
  Tables can be merged (MERGE)

  Intensity map vel choropleth 

  \vspace{-4.5mm}
  \vbox to0pt{\null\vskip3mm
